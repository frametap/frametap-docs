# Frametap Documentation

This repository contains the official documentation for [Frametap](https://frametap.io) - an API-controlled screen capture platform.

## What is Frametap?

Frametap enables developers to programmatically capture screenshots, screen recordings, and file uploads from sandboxes, containers, VMs, and remote machines. Perfect for:

- **CI/CD Pipelines** - Debug browser tests with video recordings
- **AI Agents** - Capture visual proof from sandboxed runs
- **Remote Debugging** - Record from production servers
- **Automated Monitoring** - Scheduled screenshots and artifacts

## Documentation

Visit the live documentation at [docs.frametap.io](https://docs.frametap.io)

```bash

```
